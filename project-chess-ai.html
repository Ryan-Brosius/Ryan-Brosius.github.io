Chess Engine
Developed in Python with Pygames

GitHub

I developed this basic chess engine during my summer break of freshman year of college. It was a fun project and it was my first real experience for developing my first AI. I also learned how graphics worked using the Pygames library. Overall turned out to be a fun project.

In this engine I implimented being able to play with another human, and being able to play against an AI. I first implimented the minimax algorithm and later on implimented alpha-beta pruning. I also developed a simple board ranking system to help the AI determine the best move. I also made it so the player can open up a menu to see how good the board ranking is after each move made.
