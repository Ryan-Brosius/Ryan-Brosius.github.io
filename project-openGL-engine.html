What is this rendering engine?

This openGL Engine is an engine inspired by the classic DOOM. I wanted to create a simple rendering engine to help me learn the structure of how to use OpenGL. Ever since I created my first engine which was a 2D engine, I have been really facinated with Graphics. Expecially 3D graphics!

I have learned a lot while working on this rendering engine. I learned how to use GLSL, map textures onto surfaces, how to auto triangulate simple polygons, different shading techniques, how lighting works, and much much more. Im happy with my learning progress and would like to continue working with Graphics when I graduate if possible.
