Nebula Nonsense

2D bullet hell game
Made for the 2023 Secret Santa Game Jam
Made using Unity and C#

GitHub

Itch.io

This is a 2D bullet hell game I created with my friend from college for the 2023 Secret Santa Game Jam. For this game jam teams needed to create a game for their "giftee" with a prompt given to you from said giftee. I remember our giftee wanted a space themed game. This game created here was created in around a month and was worked on during my finals week of college. Which was very fun.

The game involves three bosses, each with four different attack patterns. There is also a score system which saves and displays the best score. Below you can watch the demo of the game, look at some of the pictures of development, and play the game on itch.io!
